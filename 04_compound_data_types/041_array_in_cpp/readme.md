# array syntax in cpp

## create
~~~
element_type_name arrayName[arraySize]
~~~

## create and then assignment
~~~
int yams[3];
yams[0] = 7;
yams[1] = 8;
yams[2] = 6;
~~~
## create with initialization
~~~
int yamcost[3] = {20,30,5}  //create, initializa array
~~~

## partial initialize
~~~
float hotelTips[5] = {5.0, 2.5} //other elements set to 0
~~~
